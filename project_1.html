Quadruped Control Project

This project was done as part of MICRO 507: Legged Robots course given by Prof. Auke Jan Ijspeert and BioRob at EPFL.

Credit: Jiale Fan, Koki Kimura, Lingjun Meng.

In this project we focus on the locomotion of quadruped. We covered the problem of generating gaits using central pattern generator, discussed the topic of the reinforcement learning approach to train the robot to learn a best policy for locomotion control, and studied virtual model controller, the tegotae feedback mechanism, and came up with a simple approach to achieve omnidirectional control.

Notably, the reinforcement learning controller we trained had highest votes on best performance, and thus won the best performance award in the competition held by the Professor and TAs.

We achieve this by using our customized configuration for reinforcement learning. The keys include using linear unbounded reward for velocity along desired moving direction, adapting weights to avoid local minimum and enable gradual learning process, and pre-training on obstacle-free plane.

For more details, please turn to our report.
